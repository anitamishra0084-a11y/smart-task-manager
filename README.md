# To-Do List Application

A modern, responsive, and interactive task management application built using HTML5, CSS3, and JavaScript.

## Features
- Create, Read, Update, Delete (CRUD) tasks
- localStorage data persistence
- Task filtering (All, Active, Completed)
- Dynamic DOM manipulation
- Event delegation
- Responsive design
- Interactive user interface

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API
